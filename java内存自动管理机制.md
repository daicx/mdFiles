# java内存自动管理机制	

## 简介

java与C++：

java具有内存动态分配和垃圾回收机制。由jvm自动管理对象的地址分配和回收。正如运行在java之上的Spring框架，负责的是java对象的创建和销毁动作。

C++则需要开发人员对每一个创建的对象负责，从生到死。
